# 🏠 Luxury Housing Sales Analysis – Bengaluru

## 📌 Project Overview

This project builds an end-to-end data analytics pipeline to analyze luxury housing sales data in Bengaluru using Python, SQL, and Power BI.

---

## 🎯 Objective

* Clean and preprocess raw housing data
* Store structured data in SQL database
* Create interactive dashboards in Power BI
* Generate business insights

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* MySQL
* Power BI
* SQLAlchemy

---

## 📊 Dataset

* 100,000+ records
* Includes pricing, builder, location, and buyer details

---

## 🔄 Workflow

### 1️⃣ Data Cleaning (Python)

* Handled missing values
* Converted data types
* Feature engineering:

  * Price_per_Sqft
  * Booking_Flag
  * Quarter_Number

### 2️⃣ SQL Integration

* Created database: `luxury_housing_db`
* Loaded cleaned data into SQL
* Performed validation queries

### 3️⃣ Power BI Dashboard

* KPI Cards:

  * Total Revenue
  * Average Price
  * Booking Conversion Rate
* Visualizations:

  * Builder Performance
  * Market Trends
  * Configuration Demand
  * Amenity Impact
  * Sales Channel Analysis
  * Geographical Map

---

## 💡 Key Insights

* Top developers dominate market revenue
* 3BHK & 4BHK are most demanded
* Higher amenity scores improve booking chances
* Micro-markets show varied performance
* Booking trends vary across quarters
